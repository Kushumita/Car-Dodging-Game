# 🚗 Car Dodging Game

A simple and fun **Car Dodging Game** developed in **C++** where the player controls a car and avoids incoming obstacles to achieve the highest possible score.

## Features

* Smooth car movement controls
* Increasing difficulty over time
* Score tracking system
* Simple and interactive gameplay
* Console/graphics-based implementation in C++

## Technologies Used

* C++
* Object-Oriented Programming Concepts
* Game Logic and Collision Detection

## How to Run

### Prerequisites

Make sure you have:

* GCC/G++ compiler installed
* SFML library installed
* VS Code or any C++ IDE
* Windows operating system


### Compilation Command (Windows + SFML)

Compile the project using:

```bash
g++ CarDodging.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
```

### Run the Game

```bash
./game
```


## Gameplay

* Use keyboard controls to move the car.
* Avoid colliding with enemy cars or obstacles.
* Try to survive as long as possible to increase your score.
