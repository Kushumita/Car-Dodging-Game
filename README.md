# 🚗 Car Dodging Game (C++)

Car Dodging Game is a fun arcade-style C++ game where the player controls a car and avoids incoming enemy vehicles and obstacles. The objective is to survive as long as possible and achieve the highest score.

## 🚀 Features

* Built using C++ and SFML
* Smooth player movement
* Real-time obstacle generation
* Collision detection system
* Score tracking
* Increasing difficulty over time
* Lightweight and fast gameplay

## 🕹️ How to Play

1. Start the game
2. Control your car using keyboard keys
3. Dodge incoming enemy cars and obstacles
4. Survive as long as possible
5. Try to achieve the highest score

## 🎯 Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* ESC → Exit Game

## 🛠️ Tech Stack

* C++
* SFML Library
* Object-Oriented Programming
* Game Development Concepts

## ▶️ How to Run

### 1. Clone Repository

```bash
git clone https://github.com/Kushumita/Car-Dodging-Game.git
```

### 2. Navigate to Folder

```bash
cd Game
```

### 3. Compile

```bash
g++ CarDodging.cpp -o game -IC:/SFML/include -LC:/SFML/lib -lsfml-graphics -lsfml-window -lsfml-system
```

### 4. Run

```bash
./game
```


## 🎯 Objective

Drive carefully, avoid collisions, and survive as long as possible to achieve the highest score.
